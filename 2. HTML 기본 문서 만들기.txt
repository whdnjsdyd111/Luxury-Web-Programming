			<title>	타이틀 달기

HTML 페이지의 타이틀은 페이지의 제목으로, 브라우저의 타이틀 바에 출력된다.
타이틀은 <title> 태그를 이용하며 <head> 내에서만 작성되야 한다.

			<h1>, ... , <h6>	문단 제목(장, 절, 소제목 등)

보고서 등의 문서는 장, 절, 문단 등으로 구조화하고, 알아보기 쉽도록 장이나 절의 제목 크기를 달리한다.
HTML은 <h1>, ... , <h6>의 6개의 태그를 제공하여 문단에 제목을 붙일 수 있도록 한다.
h는 제목을 뜻하는 heading 의 줄임말이며, <h1>이 가장 큰 제목이고 <h6>가 가장 작은 제목이다.
<hn> 태그에 의해 출력되는 제목의 글자 크기는 브라우저가 결정하며, CSS 스타일 시트를 이용하여 개발자가  변경할 수 있다.

			title 속성	툴팁 달기

페이지의 본문에 마우스가 올라갈 때 설명문(툴팁)이 출력되게 할 수 있다. 툴팁(tooltip) 문자열은
태그의 title 속성에 지정하며 모든 태그는 title 속성을 가진다.

<h1 title="h1태그로 작성하였습니다.">1장 홈페이지</h1>

			<p>	단락 나누기

HTML에서 문단은 여러 단락(paragraph)으로 나눌 수 있으며 하나의 단락은 <p> 태그로 표현한다.
CSS 스타일을 사용하면 단락 단위로 내어쓰기, 들여쓰기를 제어할 수 있다. <p> 태그는 문단을 표현하기 때문에
</p>로 끝나는 문단 다음에는 자동으로 빈 줄이 생긴다.

			<hr>	수평선 긋기

문단 내에서 내용의 전환이 필요한 곳에 수평선을 삽입하여 시각적 효과를 만들 수 있다.
수평선은 <hr> 태그를 사용한다. <hr> 태그는 종료 태그가 없다.

			<br>	새로운 줄로 넘어가기

HTML 페이지에서 <Enter> 키를 여러 개 입력해도 한 개의 빈칸으로 처리되어 다음 줄로 넘어가지 않는다.
새로운 줄로 넘기고자 하면 <br> 태그를 사용해야 한다.


			문자, 기호, 심볼 입력

HTML 문서는 UTF-8 코드체계로 작성되는 것이 표준이며 UTF-8은 현재 가장 다양한 종류의 문자, 기호, 심볼을 표현하는 코드 체계이다.
웹 개발자는 이를 통해 어떤 문자라도 표현할 수 있다. 하지만 '<'나 '>'는 HTML 언어의 예약어나 
∞, ∑와 같은 키보드로 입력할 수 없는 기호, 심볼, 글자는 다음 두 방법으로 입력할 수 있다.

&엔터티; 혹은 %#코드값;

예로 다음과 같다.

<	->	&lt;	혹은 &#60;
ⓒ	->	&copy;	혹은 &#169;
∑	->	&sum;	혹은 &#8721;

이를 이용하여 기호나 심볼을 포함하는 텍스트를 다음과 같이 만들 수 있다.

2 < 3	->	2 &lt; 3
10 ÷ 2	->	10 &divide; 2
y = ∑x	->	y = &sum;x

브라우저는 연속된 여러 개의 빈칸을 한 개의 빈칸으로 처리하므로, 빈칸을 강제로 삽입하고자 하면 &nbsp; 를 삽입하면 된다.

			<pre>	개발자 포맷 그대로 출력

브라우저는 개발자가 HTML 문서의 모양을 포맷하기 위해 삽입한 여러 개의 빈칸(' ', 탭)이나 <Enter>를 모두 하나의 빈칸으로 처리하므로,
개발자가 의도한 모양대로 출력되지 않는다. 이때 <pre></pre>로 둘러싸면 개발자가 입력한 포맷 그대로 출력된다.

			텍스트 꾸미기

HTML은 강조하는 단어나, 삭제 문자를 뜻하거나, 아래 첨자나 위 첨자를 표시하는 등 텍스트에 의미를 부여할 수 있다.

<b>진하게</b>
<strong>중요한</strong>
<em>강조</em>
<i>이탤릭으로 강조</i>
<b><i>이탤릭으로 진하게 강조</i></b>
<small>한 단계 작은 문자</samll>
<del>삭제</del>
<ins>추가</ins>
<sup>위 첨자</sup>
<sub>아래 첨자</sub>
<mark>하이라이팅</mark>

			블록 태그와 인라인 태그

HTML 태그들은 블록 태그(block)와 인라인 태그(inline)로 나뉜다.

블록 태그 예 : <p>, <h1>, <div>, <ul>
인라인 태그 예: <strong>, <a>, <img>, <span>

블록 태그는 항상 새 라인에서 시작하고 브라우저의 왼쪽 끝에서 오른쪽 끝까지 블록 공간을 차지한다.
반대로 인라인 태그는 블록에 삽입되어 블록 콘텐트의 일부를 표현하는데 이용된다.	(자세한 것은 5장 1절 참고)

		<div>	많이 사용하는 블록 태그

<div>는 <p>와 함께 블록을 구성하기 위해 가장 많이 사용된다. <p> 태그는 문단을 만들기 위해 사용되지만,
<div>는 특별한 의미 없이 여러 HTML 태그들을 블록으로 묶는 컨테이너로 이용된다.
예로 <div> 블록 전체에 동일한 CSS 스타일을 적용하거나 자바스크립트로 블록을 하나의 단위처럼 다루고자 할 때이다.

		<span>	많이 사용하는 인라인 태그

<span> 태그는 텍스트 일부분에 특별한 모양을 주거나, 자바스크립트 코드로 텍스트 일부분을 제어하고자 할 때 사용한다.

			HTML 메타 데이터 삽입

메타 데이터란 데이터를 설명하는 데이터이다. 예로 사진의 경우 찍은 장소와 시간 데이터가 메타 데이터로 볼 수 있다.
HTML에서는 다음과 같은 HTML 페이지에 대한 메타 정보를 표현하는 여러 태그들을 두고 있다.

<base>, <link>, <script>, <style>, <title>, <meta>

		<base>	웹 페이지의 베이스 URL

<base> 태그는 웹 페이지들이 담겨 있는 기본 URL과 웹 페이지가 출력될 윈도우를 지정하기 위해 사용된다.
예를 들어 math.html과 science.html 웹 페이지가 http://www.mysite.com/score 에 저장되어 있다면 
<base> 태그의 href를 이용하여 베이스 URL을 다음과 같이 지정한다.

<head>
	<base href="http://www.mysite.com/score">
</head>

웹 페이지의 실제 경로명은 <base> 태그의 href 속성에 지정된 페이스 URL 에 대한 상대 경로명으로 해석된다.

<a href="math.html">수학</a>		->	http://www.mysite.com/score/math.html	로 처리

<base> 태그는 <head> 태그 내에서만 사용되며 한 번만 허용된다.

		<meta>	메타 데이터

<meta> 태그는 웹 페이지의 저작사, 문자 인코딩 방식, 문서 내용 등 다양한 메타 데이터를 표현하기 위해 사용된다.
메타 데이터는 name과 content의 속성 쌍으로 구성된다. 예로 다음과 같다.

- 웹 페이지의 저작사가 "황기태"임을 표기

<meta name="author" content="황기태">

- 웹 페이지의 내용 설명

<meta name="description" content="입학 요령에 대한 자세한 설명">

- 웹 페이지의 키워드(검색 엔진에 의해 잘 발견되도록 하기 위함)

<meta name="keywords" content="컴퓨터, 소프트웨어, 스마트폰">

- 또한 charset 속성으로 웹 페이지에 사용된 문자의 인코딩 방식을 지정할 수 있음

<meta charset="UTF-8">

UTF-8이 디폴트이므로 이 <meta> 태그는 생략해도 되지만 명시하도록 하는 것이 좋다.
